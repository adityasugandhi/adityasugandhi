### About Me

I build things that ship. Software engineer with 5+ years turning messy problems into production systems — from distributed data pipelines to AI agents that actually work.

I don't build in a vacuum. Best code I've written came from sitting next to the people who use it — facility managers, researchers, operators — and iterating until it actually solves their problem. **Ship fast, measure impact, repeat.**

- Currently building **agentic AI systems** with RAG pipelines, vector search, and LLM orchestration
- Deployed an **enterprise AI Slack bot** — answers questions and automates workflows 24/7
- Built a **building automation platform** — 37 APIs, 35K+ devices, 172 networks
